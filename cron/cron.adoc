
== JDK 中的实现
jdk 的定时器是基于 堆（基于到期时间排序）+ 阻塞队列的方式实现，java.util.Timer 和 java.util.concurrent.ScheduledExecutorService 的实现方式大同小异。
最大的区别可能是 java.util.concurrent.ScheduledExecutorService 是基于线程池实现的，而 java.util.Timer 是使用单线程进行实现。
